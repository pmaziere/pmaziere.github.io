Without google font

Default font is used as expected by firefox to display this text

With google font grabbed from <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Source+Sans+Pro%3A400%2C600%2C400italic%2C600italic&ver=4.1" type="text/css"/>

Should grab the CSS file and the fonts from the cache
  • with HTTPSEverywhere rule activated: the CSS is cached but there is no font request sent to the cache, therefore it is not used by firefox to display this text
  • without HTTPSEverywhere rule activated: the CSS and the font are downloaded and used directly from fonts.googleapis.com and fonts.gstatic.com and used as expectedby firefox to display this text
